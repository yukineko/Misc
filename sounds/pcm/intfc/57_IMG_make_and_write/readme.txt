画像を作ってRGBで書き出す．
※画像の幅は4の倍数でなければならない．
作成した画像は以下．

Bout = m;							// 青：最下段が輝度  0，最上段が輝度MAX
Gout = height - m;					// 緑：最下段が輝度MAX，最上段が輝度0
Rout = n;							// 赤：最左列が輝度  0, 最右列が輝度MAX

各色の変化を確認したい場合，他のカラーの輝度をゼロにするとよい．

例）青の確認
Bout=m;				// 青：最下段が輝度  0，最上段が輝度MAX
Gout=0;
Rout=0;



また，値が255を超えると0に戻る．つまり，
輝度256=輝度0
輝度257=輝度1
...
輝度511=輝度255
輝度512=輝度0
...

となる
